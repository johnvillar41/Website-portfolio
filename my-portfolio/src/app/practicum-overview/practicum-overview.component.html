Practicum Engagement and Presentation of Output

Accenture ATA Program

Accenture ATA Program provided 4 courses from AWS: Application Developer, Cloud Support Engineer, Introduction to Cloud, Introduction to AWS Management Console.

On each of these courses, I was tasked to watch the videos and do hands-on projects as well as knowledge checks after each videos. There were also Final exams and even projects.

The image on the left shows the topics on course Application Developer. I learned about how to develop websites using javascipt, Data structures, Operating systems, and DevOps.

The image above shows the topics on course Cloud Support Engineer. I learned about Virtualizations ond Operating Systems, Network Architecture, Programming using Python and, Security.

The last 2 courses Introduction to Cloud 101 and Introduction to AWS Management were the easiest of the 4 courses. Introduction to cloud helped me to gain knowledge and experience with cloud fundamentals especially on AWS(Amazon Web Services) Cloud while Introduction to the AWS Management Console shows us the collection of service consoles for managing AWS resources.
